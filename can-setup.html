Bosqich 1 — Ulash

Qurilma mashinaga qanday o'rnatiladi

🔧

1. Ulanish nuqtasi

OBD-II port (tez, eng oson) yoki to'g'ridan CAN simlari (CAN-H / CAN-L) ga ulanadi.

🔋

2. Quvvat

+12V, GND. Qurilma doimiy quvvatlanadi, mashina o'chsa ham uxlash rejimida ishlaydi.

📶

3. SIM + GPS

Ichida 4G/LTE SIM (internet) va GPS antenna — joylashuv uchun.

🧰

Kim o'rnatadi

Sertifikatlangan usta — 15–30 daqiqa. Bir marta.

🔒

Yashirin joy

Panel ostiga yashiriladi — o'g'rilarga ko'rinmaydi.

✅

Tekshirish

Indikator yonadi — qurilma CAN'ni o'qiy boshladi.

Natija: qurilma endi mashinaning ichki tarmog'iga "ulangan" va internetga chiqishga tayyor.

Bosqich 2 — Serverga ulanish

Qurilma bizning serverni qanday topadi

1

Yoqiladi va internetga chiqadi

SIM karta orqali mobil internetga ulanadi (operator tarmog'i).

2

O'zini tanishtiradi

Har qurilmaning noyob IMEI va maxfiy kaliti bor — server shu orqali uni taniydi.

3

Doimiy kanal ochadi

Server bilan uzluksiz aloqa o'rnatadi — komandalar darhol yetib boradi, telemetriya doimiy oqadi.

4

Ro'yxatdan o'tadi

Server bazaga yozadi: device_id · IMEI · holat: online. Endi qurilma "ko'rinadi".

Ulanishning 2 xil yo'li bor — keyingi slaydda.

Ikki integratsiya modeli

Qurilma serverga qaysi yo'l bilan ulanadi

Boshlash: StarLine (bulut API) — tez. Fleet/mustaqillik uchun: Teltonika (o'z server).

Bizning tomonimiz

Backend qanday ulanadi

Har qurilma turi uchun bitta "adapter" yoziladi. Ilova va server qismi allaqachon tayyor — yangi qurilma qo'shilsa, faqat shu adapter qo'shiladi.

// Har bir qurilma turi shu interfeysни bajaradi
type Provider interface {
  Lock(carID string) error
  Unlock(carID string) error
  StartEngine(carID string) error
  SetClimate(carID string, temp int) error
  Telemetry(carID string) (State, error)
}

// Qurilmaga qarab mos adapter tanlanadi
provider := starline.New(apiKey)  // yoki teltonika.New(...)

Ilova bir xil tugmani bosadi → server provider.Unlock() ni chaqiradi → adapter qurilma "tili"ga o'giradi. Yangi brend = yangi adapter, qolgani o'zgarmaydi.

Bosqich 3 — Sozlash

Mashina modelini kalibrlash

1

Model aniqlanadi

Masalan "Voyah Free 2024". Shu model uchun CAN xabarlari (qaysi ID nima qiladi) belgilanadi.

2

CAN profili tanlanadi

Agar profil bor bo'lsa — darhol ishlaydi. Yangi model bo'lsa — bir marta xaritalanadi ID 0x250 = eshik.

3

Komandalar test qilinadi

Lock/Unlock/Start — har biri sinaladi va tasdiqlanadi.

4

Profil saqlanadi

Endi shu modeldagi har mashina avtomatik ishlaydi — qayta kalibrlash shart emas.

Har yangi model bir marta sozlanadi — keyin kutubxonaga qo'shiladi va qayta ishlatiladi.

Bosqich 4 — Ilovaga bog'lash

Foydalanuvchi mashinani qanday qo'shadi

📲

1. Ilovani ochadi

"Mashina qo'shish" tugmasini bosadi.

🔢

2. Qurilma ID

QR-kod skanerlaydi yoki IMEI kiritadi.

✅

3. Egalik tasdiqi

SMS/kod orqali mashina egasi ekani tasdiqlanadi.

💳

4. Obuna

Obuna faollashadi — boshqaruv ochiladi.

Shu daqiqadan mashina ilovada paydo bo'ladi: holati, GPS'i va barcha tugmalari bilan. Oilaviy kirish bo'lsa — egasi boshqalarni ham qo'sha oladi.

Bosqich 5 — Ishga tushirish

Birinchi komanda

1

Foydalanuvchi "Ochish" ni bosadi

Ilova serverga xavfsiz so'rov yuboradi.

2

Server tekshiradi → qurilmaga

Ruxsat (RBAC, obuna) tekshiriladi, jurnalga yoziladi, qurilmaga uzatiladi.

3

Qurilma CAN'ga yozadi

Mos CAN frame'i shinaga yuboriladi → eshik ECU bajaradi.

4

Tasdiq qaytadi

"✓ Ochildi" ilovada ko'rinadi. Jonli telemetriya (GPS, batareya) doimiy yangilanadi.

🎉 Tayyor — mashina endi to'liq masofadan boshqariladi.
